# Hotel Booking Cancellation Prediction

Analyzed a hotel bookings dataset to predict whether a booking would be canceled or not.

## What I did
- Cleaned and explored the data to find patterns in cancellations
- Built and compared three ML models: Logistic Regression, Decision Tree, and KNN
- Evaluated using accuracy, precision, recall, and F1 score
- Used 5-fold cross validation to confirm results

## Results
- Logistic Regression: 80.6% accuracy
- Decision Tree: 85.5% accuracy (best performer)
- KNN: 77.4% accuracy

## Tools
Python, pandas, scikit-learn, matplotlib, seaborn

## Dataset
Hotel Bookings dataset with 119,390 records and 32 features
